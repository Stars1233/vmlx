6. Screenshot engine startup log for each state

### 7. No incoherency / looping / leaking / tool failure

Every response scanned for:
- Same phrase >3× in row = LOOP flag
- Leaked `<think>`, `<tool>`, `<|im_end|>`, `<im:think>`, `<parameter>`, `<function=` tags in final visible content = LEAK flag
- Empty content when reasoning fires and NO reasoning-runaway backstop
- Tool defined but not called when appropriate = TOOL_FAIL flag
- Model refuses / hallucinates schema = HALLUCINATION flag
